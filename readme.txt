Allows to use query vars to filter Query Block posts.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wi-query-tax-filter` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
